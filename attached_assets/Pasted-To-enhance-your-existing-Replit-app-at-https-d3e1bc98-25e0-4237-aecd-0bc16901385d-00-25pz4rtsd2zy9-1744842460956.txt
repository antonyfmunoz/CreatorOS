🛠️ Implementation Steps

1. Identify and Style User Tags in Text

Use JavaScript to detect @username patterns in your captions or comments and convert them into clickable links.

function parseUserTags(text) {
  return text.replace(/@(\w+)/g, '<a href="/user/$1" class="user-tag">@$1</a>');
}

Apply this function to the content before rendering it:

const captionElement = document.getElementById('caption');
const rawText = captionElement.textContent;
captionElement.innerHTML = parseUserTags(rawText);

Ensure you have a CSS class to style the user tags:

.user-tag {
  color: #3897f0;
  text-decoration: none;
  cursor: pointer;
}
.user-tag:hover {
  text-decoration: underline;
}

2. Handle Click Events on User Tags

Add an event listener to handle clicks on user tags, directing users to the appropriate profile pages.

document.addEventListener('click', function(event) {
  if (event.target.classList.contains('user-tag')) {
    event.preventDefault();
    const username = event.target.textContent.slice(1); // Remove '@'
    // Navigate to the user's profile page
    window.location.href = `/user/${username}`;
  }
});

Ensure that your application has routes set up to handle /user/:username paths, displaying the corresponding user profiles.
